BRAC (Base Refrigerante Autónoma Controlada)

Resumen

La invención trata de una base refrigerante para ordenadores portátiles, diseñada para ser autónoma y controlar su funcionamiento de acuerdo con la temperatura del dispositivo. Esta herramienta se activa automáticamente cuando la temperatura del ordenador supera un umbral específico, apagándose una vez que desciende a niveles seguros. Adicionalmente, la invención incluye un sistema de monitoreo en tiempo real que permite al usuario observar la temperatura desde su dispositivo móvil y recibir alertas en caso de sobrecalentamiento.

Antecedentes de la invención

En el mercado actual, existen diversas bases refrigerantes para ordenadores portátiles que proporcionan enfriamiento mediante ventiladores. Sin embargo, estos dispositivos carecen de la capacidad de controlar su uso de manera autónoma, lo que resulta en un consumo excesivo de energía y desgaste innecesario de las baterías de los ordenadores. Los usuarios, especialmente programadores y gamers, experimentan problemas como la reducción de la vida útil de las baterías debido a la operación constante de estos dispositivos.

Descripción detallada de la invención

La presente invención automatiza el funcionamiento de una base refrigerante para ordenadores portátiles mediante el uso de hardware adicional, incluyendo un Arduino y sensores de temperatura. El sistema monitorea continuamente la temperatura del ordenador portátil y activa los ventiladores solo cuando es necesario, reduciendo el consumo energético y prolongando la vida útil de los dispositivos. La invención incluye además un chip de conexión a internet que envía datos a una aplicación móvil o web, permitiendo al usuario supervisar el rendimiento de la base en tiempo real.

El sistema comprende:

  • Una base refrigerante ordinaria para portátiles.
  • Un Arduino para controlar los ventiladores.
  • Sensores de temperatura que determinan cuándo activar o desactivar los ventiladores.
  • Un chip para conectividad a internet, que envía información a una aplicación de monitoreo.
  • Una aplicación móvil/web para monitorear la temperatura y recibir alertas.

Reivindicaciones

  1. Un sistema de refrigeración autónoma para ordenadores portátiles, que comprende una base refrigerante equipada con:
    • Sensores de temperatura para medir la temperatura del dispositivo.
    • Un controlador (Arduino) configurado para activar y desactivar los ventiladores en función de la temperatura medida.
    • Un sistema de conexión a internet para enviar datos a una aplicación de monitoreo.
  2. El sistema de la reivindicación 1, donde la aplicación móvil/web permite al usuario monitorear la temperatura del ordenador y recibir alertas cuando la base refrigerante se activa.
  3. El sistema de la reivindicación 1, donde la activación de los ventiladores se realiza solo cuando la temperatura del ordenador portátil supera un umbral predefinido, y se desactiva cuando desciende por debajo de este umbral.

Dibujos o esquemas

Se pueden incluir esquemas del circuito de Arduino con los sensores de temperatura, el módulo de internet y la disposición de los ventiladores en la base refrigerante.

Descripción de los dibujos

  1. Diagrama Metodologia implementada para la construccion.


  2. Diagrama de la conexion interna del dispositivo.


  3. Representación real de la interfaz en app mobile y ordenador portatil montaje en la base refrigerante.

Ejemplos o realizaciones preferidas

En una realización preferida, la base refrigerante utiliza sensores de temperatura ubicados estratégicamente cerca de los puntos de mayor generación de calor del ordenador portátil. El sistema activa los ventiladores cuando la temperatura alcanza los 75°C y los desactiva cuando baja a 60°C. La aplicación móvil notifica al usuario cuando se producen estos eventos y ofrece un historial de temperaturas para análisis posterior.

Información técnica y científica adicional

El desarrollo se basó en la metodología de Design Thinking, utilizando fases como empatía, definición, ideación, prototipado y evaluación. Se integraron tecnologías de hardware como Arduino y ESP8266 para la conectividad a internet, y una aplicación móvil para la interfaz de usuario.

Conclusiones

La presente invención proporciona una solución efectiva y eficiente para los problemas de sobrecalentamiento de ordenadores portátiles, reduciendo el consumo energético y el desgaste de las baterías. La automatización del encendido y apagado de los ventiladores, junto con el sistema de monitoreo en tiempo real, ofrece una mejora significativa sobre las bases refrigerantes convencionales.
